# Separate columns (base)

This is a utility function used to create the list of applications in
the Git branches.

## Usage

``` r
sep_cols(data, col, into, sep)
```

## Arguments

- data:

  data.frame or tibble

- col:

  column to split

- into:

  columns to split into

- sep:

  regex to split on

## Value

data with new columns
